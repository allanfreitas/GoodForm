Radio Input

There are two ways of adding a single radio field to your form object:

radio (string name [, string value, mixed checked])

  • name: defines the elements name attribute, should be unique in the object | required
  • value: defines the default field value | optional
  • checked: defines the checked status of the field value | optional

You can flag a radio button as checked by passing non-empty value to the checked parameter or in the checked attribute of the array parameter.
e.g. TRUE, 1, 'checked' or "lollipops"

Example:

$this->goodform->radio('sex', 'male', TRUE);

Will produce the following HTML:

<input name="sex" value="male" checked="checked" type="radio" />

radio (array attributes)

  • attributes: an associative array specifying attributes of the field element.

All attributes in this array will be converted to HTML element attributes. GoodForm will not check if these attributes are HTML valid. The only exceptions to this are the Special attributes that are not passed to the elements attribute string.

Example:

$attr = array(
	'name' 		=> 'sex',
	'value'		=> 'female',
	'checked'	=> 'checked'
	'class'		=> 'my-radio-class'
);

$this->goodform->radio($attr);

Will produce the following HTML:

<input name="sex" value="female" class="my-radio-class" checked="checked" type="radio" />

Radio Groups

Goodform also provides methods for adding groups of radie buttons. There are two ways to add a radio group to your form.

radio_group (string name [, array options, mixed checked])

  • name: defines the elements name attribute, should be unique in the object. Goodform will automatically add a [] to each options name. | required
  • options: defines the groups radio options. Array should specify label => value to allow for multiple options with the same value. | required
  • checked: a single value that specifies which radio option is selected. | optional

Example:

$stations = array(
	'none'			=> NULL,
	'Radio 1'		=> 1,
	'Radio 2'		=> 2,
	'Radio 3'		=> 3,
	'Radio 4'		=> 4,
	'Radio 5 Live'	=> 5,
	'Radio 6 Music'	=> 6,
);
		
$this->goodform->radio_group('fav_station', $stations, 4);

Will produce the following HTML:

<input value="" name="fav_station[]" type="radio"/>
	<span class="label">none</span>
	<input value="1" name="fav_station[]" type="radio"/>
	<span class="label">Radio 1</span>
	<input value="2" name="fav_station[]" type="radio"/>
	<span class="label">Radio 2</span>
	<input value="3" name="fav_station[]" type="radio"/>
	<span class="label">Radio 3</span>
	<input value="4" name="fav_station[]" type="radio" checked="checked"/>
	<span class="label">Radio 4</span>
	<input value="5" name="fav_station[]" type="radio"/>
	<span class="label">Radio 5 Live</span>
	<input value="6" name="fav_station[]" type="radio"/>
	<span class="label">Radio 6 Music</span>
	<div class="clear"></div>
</div>

radio_group (array attributes)

  • attributes: an associative array specifying attributes of the field element.

The 'direction' attribute is used to specify the layout of the input options. Set to 'h' or 'v' (default).

Example:

$attr = array(
	'name' => 'fav_ice_cream',
	'checked' => 3,
	'options' => array(
		'Vanilla' => 0,
		'Chocolate' => 1,
		'Strawberry' => 2,
		'Coffee' => 3
	),
	'class' => 'vert-input-group'
);
		
$this->goodform->radio_group($attr);

Will produce the following HTML:

<div class="input-group vert-input-group">
	<input value="0" name="fav_ice_cream[]" type="radio"/>
	<span class="label">Vanilla</span>
	<input value="1" name="fav_ice_cream[]" type="radio"/>
	<span class="label">Chocolate</span>
	<input value="2" name="fav_ice_cream[]" type="radio"/>
	<span class="label">Strawberry</span>
	<input value="3" name="fav_ice_cream[]" type="radio" checked="checked"/>
	<span class="label">Coffee</span>
	<div class="clear"></div>
</div>

Radio Groups with Option Groups

You can spoof select box option groups by supplying a 2D option array. Each group will be divided by a label.

$attr = array(
	'name' => 'fav_ice_cream',
	'checked' => 3,
	'options' => array(
		'Vanilla' => array(
			'On its own' =>  1, 
			'And Choc Chips' => 2,
		),
		'Chocolate' => array(
			'On its own' => 3,
			'And Mint' => 4,
			'And Toffee' => 5		
		),
		'Strawberry' => 6,
		'Coffee' => 7
	)
);
		
$this->goodform->radio_group($attr);

Will produce the following HTML:

<div class="optgroup">
		<p class="optgroup-label">Vanilla</p>
		<div class="input-group">
			<input value="1" name="fav_ice_cream[]" type="radio"/><span class="label">On its own</span>
			<input value="2" name="fav_ice_cream[]" type="radio"/><span class="label">And Choc Chips</span>
			<div class="clear"></div>
		</div>
	</div>
	<div class="optgroup">
		<p class="optgroup-label">Chocolate</p>
		<div class="input-group">
			<input value="3" name="fav_ice_cream[]" type="radio" checked="checked"/><span class="label">On its own</span>
			<input value="4" name="fav_ice_cream[]" type="radio"/><span class="label">And Mint</span>
			<input value="5" name="fav_ice_cream[]" type="radio"/><span class="label">And Toffee</span>
			<div class="clear"></div>
		</div>
	</div>
	<input value="6" name="fav_ice_cream[]" type="radio"/><span class="label">Strawberry</span>
	<input value="7" name="fav_ice_cream[]" type="radio"/><span class="label">Coffee</span>
	<div class="clear"></div>
